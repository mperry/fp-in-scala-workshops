Functions are Values

  • Can be passed as arguments
  • Stored in variables, data structures
  • Loop without mutation


def factorial(n: Int): Int = {
	def go(n: Int, acc: Int): Int =
		if (n <= 0) acc
		else go(n-1, n*acc)
	go(n, 1)
}

Recursion


def factorial(n: Int): Int = {
	def go(n: Int, acc: Int): Int =
		if (n <= 0) acc
		else go(n-1, n*acc)
	go(n, 1)
}
  • Recursive calls implement loop
  • Here recursive call happens last (tail position)
  • Tail call optimisation (TCO) changes recursion to while loop
